Template String provide an easy way to interpolate variables and expressions into strings

```js
using regular string, ""
let a = "Hello"
let b = "World"
console.log(a + " " + b)
```

using Tamplate Strings, also known as back ticks
``
```js
console.log(`${a} ${b}`)
```

any javascript code can be kept inside `${}`

```js
console.log(`${2 + 3} ${b}`)
```
