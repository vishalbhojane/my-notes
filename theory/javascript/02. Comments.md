## Single-line Comments

```javascript
// This is a single-line comment
let x = 5; // End-of-line comment
```

## Multi-line Comments

```javascript
/*
 Multi-line comment
 Spans several lines
*/

/**
 * Documentation
 * style comment
 */
```

## VS Code Shortcut

Toggle single-line: `Ctrl + /` (Windows/Linux) or `Cmd + /` (Mac)
Toggle block comment: `Shift + Alt + A` (Windows/Linux) or `Shift + Option + A` (Mac)
